**Ты запускаешь программу, а она молчит. Зависла? Работает? Сколько ещё ждать?**  
tqdm-demonstrator показывает, как заставить консоль говорить с тобой на понятном языке.

---

## ❓ Зачем это нужно?

| Проблема | Как решает tqdm-demonstrator |
|----------|------------------------------|
| Ты запустил обработку 1000 файлов и не знаешь, сколько ждать | Программа показывает **точно**: 45% готово, осталось 23 секунды |
| Кажется, что программа зависла, хотя она просто думает | Бегущая полоска доказывает: **всё идёт по плану** |
| Нужно сравнить, какой алгоритм быстрее | tqdm показывает **скорость обработки** в реальном времени |


---

## ✨ Возможности

✅ **Три живых сценария** — файлы, сеть, вычисления. Каждый показывает свою фишку tqdm  
✅ **Вложенные прогресс-бары** — видно и общий прогресс, и прогресс внутри каждого файла  
✅ **Менеджер хранилища** — просмотр, удаление, архивация файлов (никакого мусора на диске)  
✅ **Цветной интерфейс** — приятно смотреть, удобно читать  
✅ **Живая статистика** — скорость, оставшееся время, успешные/ошибочные операции  
✅ **Полный контроль** — сам решаешь, что удалять, а что оставить

## 🚀 Быстрый старт (за 30 секунд)

### Для обычных пользователей (рекомендуется)
1. Скачай Python с [официального сайта](https://python.org) (если ещё нет)
2. Скачай проект [отсюда](https://github.com/Mateush2131/trqdm-demonstrator)
3. Запусти `run.py` двойным кликом
4. Выбирай сценарий и смотри магию
